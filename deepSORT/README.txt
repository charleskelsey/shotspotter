deepSORT implementation source: https://github.com/levan92/deep_sort_realtime

This script takes the yolo output csv file as input. You need access to the source video as well as the yolo output. To run install dependencies and add source video to script directory. change 'VIDEO_PATH' variable to appropriate name. Ensure 'detections.csv' file is the output of the yolo model with same source video as input.
